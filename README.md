# FakeStore App

This is a simple React app created with Vite. It shows products from FakeStoreAPI and lets you:

- see a home page
- view a list of products
- open product details
- add a product
- edit a product

## How to run it

1. Open a terminal in the project folder.
2. Install dependencies:

```bash
npm install
```

3. Start the app:

```bash
npm run dev
```

4. Open the address shown in the terminal (usually `http://localhost:5174`).

## What is inside

- `src/App.jsx`: sets up the app routes
- `src/pages/`: contains the different pages
- `src/components/`: contains reusable components like the navbar
- `src/index.css`: contains the app styles
